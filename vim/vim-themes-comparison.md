# Ron vs Cyberpunk Theme Comparison

| Element | Ron | Cyberpunk |
|---|---|---|
| **Normal text** | Cyan (`#00ffff`) | Light gray (`#d3d3d3`) |
| **Comments** | Green (`#00ff00`) | Gray, italic (`#8B8989`) |
| **Strings** | Cyan (inherits Constant) | Green (`#61CE3C`) |
| **Keywords/Statements** | Light blue (`#add8e6`) | Blue (`#4c83ff`) |
| **Functions** | Cyan (inherits Identifier) | Deep pink (`#ff1493`) |
| **Identifiers** | Cyan (`#00ffff`) | Hot pink (`#ff69b4`) |
| **Types** | Sea green, bold (`#2e8b57`) | Light green (`#afd8af`) |
| **Constants/Numbers** | Cyan, bold | Light blue (`#96CBFE`) |
| **Operators** | Orange (`#ffa500`) | Cyan (`#00ffff`) |
| **PreProc** | Pink (`#eea9b8`) | Gray (`#919191`) |
| **Visual selection** | Reverse video | Dark magenta bg (`#7F073F`) |
| **Search** | Black on gray | Black on yellow |
| **IncSearch** | Steel blue bg | Deep pink bg |
| **StatusLine** | Cyan on blue | Blue on dark gray |
| **LineNr** | Gray (`#a9a9a9`) | Muted green (`#9fc59f`) |
| **MatchParen** | Dark cyan bg | Deep pink bg |

Ron is heavily cyan-dominated — normal text, identifiers, constants, strings, and directory names are all the same cyan, making it hard to distinguish syntax elements. Cyberpunk uses a wider palette with distinct colors per syntax role (pink functions, green strings, blue keywords, light blue constants), giving much more visual differentiation.

## Local ron.vim vs System ron.vim

The local `vim/ron.vim` is the original minimal version by Ron Aaron. The system version (`/usr/share/vim/vim90/colors/ron.vim`) is a modernized rewrite generated by Colortemplate that adds cterm fallbacks for 256/16/8 color terminals and defines more highlight groups.

The core syntax colors are identical between the two. The only differences are in the Diff highlight groups:

| Diff group | Local ron.vim | System ron.vim |
|---|---|---|
| **DiffAdd** | slateblue bg | `#5F875F` (muted green) bg |
| **DiffChange** | darkgreen bg | `#5F87AF` (steel blue) bg |
| **DiffDelete** | coral bg | `#AF5FAF` (purple) bg |
| **DiffText** | olivedrab bg | `#C6C6C6` (light gray) bg |
